# Задача: Выбор модели управления проектом

Для выбранного проекта необходимо:

- **Модель управления:** указать выбранную модель в поле «Модель управления».
- **Обоснование:** кратко объяснить выбор модели и расписать этапы разработки согласно ей.
- **Тип приложения:** выбрать один или несколько типов приложений для реализации проекта.
- **Компоненты продукта:** определить составные части (компоненты) разрабатываемого решения.
- **Команда и вовлечённость:** сформировать команду, указать участников и их роли на каждом этапе разработки.
