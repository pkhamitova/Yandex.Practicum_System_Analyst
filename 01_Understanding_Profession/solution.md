# Решение задачи: Выбор модели управления проектом
В этом документе представлено решение задачи по выбору модели управления для проекта. Основные выводы:

*   **Выбранная модель:** Каскадная модель (Waterfall).
*   **Обоснование:** Модель выбрана из-за четкой и измеримой цели, простоты функционала и жестких временных рамок. Для медицинской сферы критически важна полная техническая документация на выходе, которую гарантирует Waterfall. Гибкие методологии (Agile) избыточны для этого объема работ, так как не предполагается итеративная доработка или масштабирование продукта.
*   **Тип приложения:** Встроенное приложение (ПО для медицинского оборудования).
*   **Ключевые компоненты:** Клиентская часть (интерфейс для врача), Серверная логика (обработка данных остроты зрения), База данных (хранение показателей пациентов и нормативных значений).
*   **Команда:** Системный аналитик, Backend-разработчик, Frontend-разработчик, Тестировщик, Проджект-менеджер, Специалист сопровождения.
